# Path_finding
Dijkstra and Astar in Julia language 


Pour lancer les 2 algorithme:

1) Se placer dans le dossier ou se situe le fichier pathfinding
2) include le fichier julia: include("pathfinding.jl")
3) se placer dans le dossier street_map pour avoir accès au maps.
4) appeler l'algorithme souhaiter de la façon suivante: Astar("Nom_de_map.map",(x,y),(z,w)) avec départ=(x,y) et arrivée=(z,w)
5) Pour dijkstra marche de la même façon en remplaçant Astar par Dijkstra
